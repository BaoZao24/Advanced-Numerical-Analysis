# 2.11 二次重节点插值多项式及余项估计

## 题目要求
用插值方法求在 $x=0$ 处与 $\cos x$ 相切，在 $x=\frac{\pi}{2}$ 处与 $\cos x$ 相交的二次多项式 $p_2(x)$，并推导插值余项的估计式为：
$$ |\cos x - p_2(x)| \le \frac{1}{6} x^2 \left|x - \frac{\pi}{2}\right| $$

---

## 知识点解析

这些知识点主要用于根据已知节点或函数条件构造插值多项式，并进一步估计误差、分析插值性质或计算函数近似值。做题时通常先判断适合哪一种插值形式，再构造多项式，最后结合余项或性质完成证明与估计。

### 1. Hermite (带导数条件) 插值
当插值条件不仅要求函数值相等，还要求导数值相等时，这被称为 Hermite 插值。
- “在 $x=0$ 处与 $f(x)$ 相切” 意味着：插值多项式在 $x=0$ 处的函数值和一阶导数值，都与原函数相等。即 $p_2(0) = f(0)$ 且 $p_2'(0) = f'(0)$。此时 $x=0$ 是一个**二重插值节点**。
- “在 $x=\frac{\pi}{2}$ 处与 $f(x)$ 相交” 意味着：在此处的函数值相等。即 $p_2(\frac{\pi}{2}) = f(\frac{\pi}{2})$。此时 $x=\frac{\pi}{2}$ 是一个**单重插值节点**。

### 2. 插值余项定理扩展
对于具有重节点的插值多项式，其余项公式的节点多项式部分会根据重数增加相应的幂次。这里 $x=0$ 是二重节点，$x=\frac{\pi}{2}$ 是单重节点，共需要 3 个条件，因此可以唯一确定一个次数不超过 2 的多项式。
其截断误差（余项）可以表示为：
$$ R_2(x) = \frac{f^{(3)}(\xi)}{3!} (x - 0)^2 \left(x - \frac{\pi}{2}\right) $$
其中 $\xi$ 在包含 $x, 0, \frac{\pi}{2}$ 的区间内。

### 3. 重节点对插值多项式 $p(x)$ 的影响（拓展）
很多时候会有疑问：节点多项式增加幂次，是不是只对“余项”有效，对 $p(x)$ 不有效？
实际上，重节点对 $p(x)$ 的构造同样有至关重要的影响，只是在不同求解方法中表现形式不同：
- **待定系数法（本文所用）：** 直接设 $p_2(x) = ax^2 + bx + c$。重节点的“幂次”隐式地体现在它提供了额外的**导数条件**（如二重节点提供一阶导，三重节点提供一阶和二阶导等）。重数越高，提供的导数阶数约束越多。
- **牛顿插值法（Newton 差商）：** 如果我们用牛顿基本公式来构造 Hermite 插值多项式，乘方项是普遍存在的规律，**不仅限于二重节点！** 
  对于任意 $m$ 重节点 $x_0$，在差商表中会将 $x_0$ 连续记录 $m$ 次。对应的牛顿多项式展开将会自然生成一系列递增的幂次：
  $$ p(x) = f(x_0) + f[x_0,x_0](x - x_0) + f[x_0,x_0,x_0](x - x_0)^2 + \dots + f[\underbrace{x_0,\dots,x_0}_{m个}](x - x_0)^{m-1} + \dots $$
  就本题（二重节点 $x=0$，即 $m=2$）而言，差商表会把 $0$ 写两次，$p_2(x)$ 的表达式就写成了：
  $$ p_2(x) = f(0) + f[0,0](x - 0) + f\left[0,0,\frac{\pi}{2}\right](x - 0)^2 $$
  可以清楚地看到，**多项式 $p(x)$ 的基函数中会伴随节点重数自然生长出相应的幂次项**。

因此，重节点的规律不仅仅控制着余项的次数，也完全主导着 $p(x)$ 自身的代数构造结构。

### 4. 关于余项记号 $R_n(x)$ 的易错点说明
很多时候容易误以为 $R_2(x)$ 代表“最高幂次数为 2”，这是不准确的。
- **角标含义：** $R_n(x)$ 中的下标 $n$ 是为了与插值多项式 $p_n(x)$ 对应，表示这是**“逼近一个 $n$ 次多项式所产生的余项”**。
- **实际幂次：** 根据余项定理公式 $R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!} \omega_{n+1}(x)$。这里的 $\omega_{n+1}(x)$ 是由所有插值节点连乘得到的。由于一个 $n$ 次多项式需要 $n+1$ 个插值条件，因此 $\omega_{n+1}(x)$ 其实是一个 **$n+1$ 次** 的多项式。
- 对比本题：我们求的是二次多项式 $p_2(x)$，所以余项记作 $R_2(x)$。但因为插值条件有 3 个（$x=0$ 算两个，$x=\frac{\pi}{2}$ 算一个），其节点多项式部分为 $x^2(x - \frac{\pi}{2})$，展开后最高次幂实际上是 **3次方**！

## 解题与推导过程

**第一部分：求二次插值多项式 $p_2(x)$**

已知目标函数 $f(x) = \cos x$。我们先求出相关条件：
- $f(0) = \cos 0 = 1$
- $f'(0) = -\sin 0 = 0$
- $f\left(\frac{\pi}{2}\right) = \cos\left(\frac{\pi}{2}\right) = 0$

**解法一：待定系数法**
设二次多项式为 $p_2(x) = ax^2 + bx + c$，$p_2'(x) = 2ax + b$。根据条件：
1. $p_2(0) = c = 1$
2. $p_2'(0) = b = 0$
3. $p_2\left(\frac{\pi}{2}\right) = a\left(\frac{\pi}{2}\right)^2 + c = 0 \Rightarrow a = -\frac{4}{\pi^2}$
得到该二次插值多项式为：$p_2(x) = 1 - \frac{4}{\pi^2}x^2$。

**解法二：Newton 差商法（体现重节点特性）**
因为 $x=0$ 是二重节点，我们在构造差商表时，将 $0$ 连续记录两次。遇到重节点求一阶差商时，直接用函数的导数值代替：
| $x_i$ |   $f(x_i)$   |  一阶差商 $f[\cdot,\cdot]$  | 二阶差商 $f[\cdot,\cdot,\cdot]$ |
|:-----:|:------------:|:-------------------------:|:------------------------------:|
|   $0$   |      $1$       |                           |                                |
|   $0$   |      $1$       | $f[0,0] = f'(0) = 0$      |                                |
| $\frac{\pi}{2}$ | $0$ | $f[0,\frac{\pi}{2}] = \frac{0 - 1}{\frac{\pi}{2} - 0} = -\frac{2}{\pi}$ | $f[0,0,\frac{\pi}{2}] = \frac{-\frac{2}{\pi} - 0}{\frac{\pi}{2} - 0} = -\frac{4}{\pi^2}$ |

根据 Newton 插值公式：
$$ p_2(x) = f(0) + f[0,0](x - 0) + f\left[0,0,\frac{\pi}{2}\right](x - 0)^2 $$
代入差商表对角线上求得的值：
$$ p_2(x) = 1 + 0 \cdot x + \left(-\frac{4}{\pi^2}\right) x^2 $$
$$ p_2(x) = 1 - \frac{4}{\pi^2}x^2 $$
Newton 法非常直观地展现了上文提到的：二重节点会让 $p(x)$ 自然长出 $(x-0)^2$ 这项规律。

**解法三：Hermite 插值基函数构造法**
针对题目给定的特定条件（$x=0$ 为二重节点，$x=\frac{\pi}{2}$ 为一重节点），我们也可以直接利用 Hermite 插值的思想构造规范基函数。
设 $p_2(x) = f(0)\alpha_0(x) + f'(0)\beta_0(x) + f\left(\frac{\pi}{2}\right)\alpha_1(x)$。
由于本题中 $f'(0)=0$ 且 $f\left(\frac{\pi}{2}\right)=0$，后面的两项其实全都消掉了！最终多项式 $p_2(x)$ 实际上完全被 $f(0)$ 的基函数决定，即 $p_2(x) = f(0)\cdot \alpha_0(x) = 1 \cdot \alpha_0(x)$。
因此我们只需要纯粹地构造出一个二次多项式 $\alpha_0(x)$，它必须满足自身的规范条件：
- $\alpha_0(0) = 1$ （在目标点值为 1）
- $\alpha_0'(0) = 0$ （在二重节点的导数为 0）
- $\alpha_0\left(\frac{\pi}{2}\right) = 0$ （在其他节点的值为 0）

已知它的导数为 0 说名它在 $x=0$ 处取得极值（对称轴为 $y$ 轴），由此可写出形式 $\alpha_0(x) = kx^2 + 1$。
再代入另一个节点的零点条件：$k\left(\frac{\pi}{2}\right)^2 + 1 = 0 \Rightarrow k = -\frac{4}{\pi^2}$。
从而基函数 $\alpha_0(x) = 1 - \frac{4}{\pi^2}x^2$。
最终 $p_2(x) = \alpha_0(x) = 1 - \frac{4}{\pi^2}x^2$。三种方法殊途同归，结果完全一致！

**第二部分：推导插值余项估计式**

由于 $p_2(x)$ 满足三个插值条件（$x=0$ 为二重节点，$x=\pi/2$ 为单节点），根据 Hermite 插值余项公式：
$$ R_2(x) = f(x) - p_2(x) = \frac{f^{(3)}(\xi)}{3!} (x - 0)^2 \left(x - \frac{\pi}{2}\right) $$

我们需要计算原函数 $f(x) = \cos x$ 的三阶导数：
- $f'(x) = -\sin x$
- $f''(x) = -\cos x$
- $f^{(3)}(x) = \sin x$

代入余项公式可得：
$$ \cos x - p_2(x) = \frac{\sin \xi}{6} x^2 \left(x - \frac{\pi}{2}\right) $$

对等式两边取绝对值进行估计：
$$ |\cos x - p_2(x)| = \left| \frac{\sin \xi}{6} x^2 \left(x - \frac{\pi}{2}\right) \right| $$

因为对于任意实数 $\xi$，正弦函数的绝对值有上界 $|\sin \xi| \le 1$，且 $x^2 \ge 0$，因此我们可以放缩：
$$ |\cos x - p_2(x)| \le \frac{1}{6} x^2 \left|x - \frac{\pi}{2}\right| $$

**证毕。**
