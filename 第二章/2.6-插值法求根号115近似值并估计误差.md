# 2.6 插值法求根号 115 的近似值并估计误差

## 题目

利用插值方法求

```math
\sqrt{115}
```

的近似值，并估计误差。

---

## 这题要做什么

要求的是 `\sqrt{115}`，可以把它看成函数

```math
f(x)=\sqrt{x}
```

在 `x=115` 处的函数值。

也就是要求

```math
f(115)=\sqrt{115}
```

插值法的思想是：

先找几个容易计算的点，例如完全平方数附近的点，然后用一个简单多项式近似 `f(x)=\sqrt{x}`。

因为

```math
100<115<121
```

并且

```math
\sqrt{100}=10,\quad \sqrt{121}=11
```

所以最自然地选取插值节点

```math
x_0=100,\quad x_1=121
```

然后用一次插值多项式近似 `\sqrt{x}`。

---

## 用到的知识点

### 1. 函数插值思想

如果函数 `f(x)` 在若干个节点上的函数值已知，就可以构造一个多项式 `p(x)`，使它在这些节点上与 `f(x)` 取相同函数值。

然后用 `p(x)` 近似 `f(x)`。

本题中：

```math
f(x)=\sqrt{x}
```

我们用一次插值多项式 `p_1(x)` 来近似它。

---

### 2. 一次 Lagrange 插值公式

对于两个节点 `x_0,x_1`，一次 Lagrange 插值多项式为

```math
p_1(x)=f(x_0)\frac{x-x_1}{x_0-x_1}+f(x_1)\frac{x-x_0}{x_1-x_0}
```

---

### 3. 插值余项公式

一次插值的误差公式为

```math
R_1(x)=f(x)-p_1(x)=\frac{f''(\xi)}{2!}(x-x_0)(x-x_1)
```

其中 `\xi` 在 `x_0` 和 `x_1` 之间。

因此误差估计为

```math
|R_1(x)|\le \frac{M_2}{2}|(x-x_0)(x-x_1)|
```

其中

```math
M_2=\max_{x\in[x_0,x_1]}|f''(x)|
```

---

## 解题步骤

### 第一步：选取函数和插值节点

令

```math
f(x)=\sqrt{x}
```

因为 `115` 靠近完全平方数 `100` 和 `121`，所以取

```math
x_0=100,\quad x_1=121
```

对应函数值为

```math
f(100)=\sqrt{100}=10
```

```math
f(121)=\sqrt{121}=11
```

---

### 第二步：写出一次插值多项式

由一次 Lagrange 插值公式，得

```math
p_1(x)=10\cdot\frac{x-121}{100-121}+11\cdot\frac{x-100}{121-100}
```

即

```math
p_1(x)=10\cdot\frac{x-121}{-21}+11\cdot\frac{x-100}{21}
```

也可以写成

```math
p_1(x)=10\cdot\frac{121-x}{21}+11\cdot\frac{x-100}{21}
```

整理：

```math
p_1(x)=\frac{10(121-x)+11(x-100)}{21}
```

```math
p_1(x)=\frac{1210-10x+11x-1100}{21}
```

```math
p_1(x)=\frac{x+110}{21}
```

---

### 第三步：代入 `x=115`

于是

```math
\sqrt{115}=f(115)\approx p_1(115)
```

```math
p_1(115)=\frac{115+110}{21}
```

```math
p_1(115)=\frac{225}{21}=\frac{75}{7}
```

```math
p_1(115)\approx 10.7142857
```

所以

```math
\sqrt{115}\approx 10.7143
```

---

## 误差估计

由一次插值余项公式：

```math
R_1(x)=f(x)-p_1(x)=\frac{f''(\xi)}{2}(x-100)(x-121)
```

其中 `\xi\in(100,121)`。

因为

```math
f(x)=\sqrt{x}=x^{1/2}
```

所以

```math
f'(x)=\frac{1}{2}x^{-1/2}
```

```math
f''(x)=-\frac{1}{4}x^{-3/2}
```

因此

```math
|f''(x)|=\frac{1}{4x^{3/2}}
```

在区间 `[100,121]` 上，`|f''(x)|` 随 `x` 增大而减小，所以最大值在 `x=100` 处取得：

```math
M_2=|f''(100)|=\frac{1}{4\cdot 100^{3/2}}
```

因为

```math
100^{3/2}=1000
```

所以

```math
M_2=\frac{1}{4000}
```

于是当 `x=115` 时，误差满足

```math
|R_1(115)|\le \frac{1}{2}\cdot \frac{1}{4000}\cdot |(115-100)(115-121)|
```

```math
|R_1(115)|\le \frac{1}{8000}\cdot |15\cdot (-6)|
```

```math
|R_1(115)|\le \frac{90}{8000}
```

```math
|R_1(115)|\le 0.01125
```

所以误差估计为

```math
\boxed{|\sqrt{115}-10.7143|\le 0.01125}
```

---

## 误差正负的判断

因为

```math
f''(x)=-\frac{1}{4x^{3/2}}<0
```

所以 `f(x)=\sqrt{x}` 是上凸函数。

对于上凸函数，连接两端点的弦在函数图像下方，故一次插值值偏小。

因此

```math
p_1(115)<\sqrt{115}
```

也就是说

```math
\sqrt{115}>10.7143
```

结合误差估计可得

```math
10.7143<\sqrt{115}<10.7143+0.01125
```

即

```math
10.7143<\sqrt{115}<10.72555
```

---

## 最终答案

利用一次插值法，取节点

```math
x_0=100,\quad x_1=121
```

得到

```math
p_1(x)=\frac{x+110}{21}
```

所以

```math
\sqrt{115}\approx p_1(115)=\frac{75}{7}\approx 10.7143
```

误差估计为

```math
|\sqrt{115}-10.7143|\le 0.01125
```

即

```math
\boxed{\sqrt{115}\approx 10.7143}
```

```math
\boxed{|误差|\le 0.01125}
```

---

## 这类题怎么做

以后遇到“用插值法求根号的近似值”这类题，可以按下面步骤做：

### 第一步：把根号写成函数值

例如

```math
\sqrt{115}=f(115),\quad f(x)=\sqrt{x}
```

### 第二步：找附近容易开方的数

找离 `115` 最近的完全平方数：

```math
100=10^2,\quad 121=11^2
```

所以取节点 `100` 和 `121`。

### 第三步：写一次插值多项式

用公式

```math
p_1(x)=f(x_0)\frac{x-x_1}{x_0-x_1}+f(x_1)\frac{x-x_0}{x_1-x_0}
```

### 第四步：代入要求的点

算

```math
p_1(115)
```

它就是 `\sqrt{115}` 的近似值。

### 第五步：用余项公式估计误差

一次插值误差公式是

```math
R_1(x)=\frac{f''(\xi)}{2}(x-x_0)(x-x_1)
```

求出 `|f''(x)|` 在插值区间上的最大值，就可以估计误差。
