# 2.13 差商（均差）与多项式次数的关系

## 题目要求
填空题计算差商的值：
(1) 若 $f(x) = 4x^2 + x - 3$，$x_k = kh \ (k=0,1,\dots)$，则 $f[x_n, x_{n+1}, x_{n+2}] = \ \underline{\quad}\ $；$f[x_n, x_{n+1}, x_{n+2}, x_{n+3}] = \ \underline{\quad}\ $。
(2) 若 $f(x) = x^7 + x^4 + 3x + 1$，则 $f[2^0, 2^1, \dots, 2^7] = \ \underline{\quad}\ $；$f[2^0, 2^1, \dots, 2^8] = \ \underline{\quad}\ $。
(3) 若 $f(x) = 3x^2 + 1$，则 $f[1,2,3] = \ \underline{\quad}\ $；$f[1,2,3,4] = \ \underline{\quad}\ $。

---

## 知识点解析

这些知识点主要用于根据已知节点或函数条件构造插值多项式，并进一步估计误差、分析插值性质或计算函数近似值。做题时通常先判断适合哪一种插值形式，再构造多项式，最后结合余项或性质完成证明与估计。

### 1. 差商的阶数与节点个数
差商的阶数（阶次）等于括号内**节点的个数减去 1**。
- 例如 $f[x_0, x_1, x_2]$ 里面有 3 个点，它叫**二阶差商**。
- 例如 $f[2^0, 2^1, \dots, 2^7]$ 里面有从 $2^0$ 到 $2^7$ 共 8 个点，它是**七阶差商**。

### 2. 差商与导数的关系（核心定理！）
差商与导数之间存在一个极其重要的等价定理：
若 $f(x)$ 在包含所有插值节点的区间内有直至 $n$ 阶连续导数，则 $n$ 阶差商可以表示为：
$$ f[x_0, x_1, \dots, x_n] = \frac{f^{(n)}(\xi)}{n!} $$
其中 $\xi$ 是区间内的某个值。

### 3. 多项式的差商性质（终极秒杀技巧）
如果 $f(x) = a_n x^n + \dots + a_1 x + a_0$ 恰好是一个最高次为 $n$ 次的多项式，那么对其不断求导有：
- **求 $n$ 阶导数：** 最高次项 $a_n x^n$ 求 $n$ 次导数后变成了常数 $n! \cdot a_n$，后面的低次项全都求导变成了 0。所以 $f^{(n)}(x) \equiv n! \cdot a_n$。
- **求高于 $n$ 阶的导数：** 常数继续求导得 0，所以 $f^{(k)}(x) \equiv 0 \quad (k > n)$。

结合“知识点2”的定理，我们能立刻得出以下无敌推论：对于最高次数为 $n$ 的多项式：
1. 它的 **$n$ 阶差商**恒等于一个常数，且刚好等于其**最高次项系数 $a_n$**。
2. 它的 **高于 $n$ 阶的差商**恒等于 **$0$**。

---

## 解题过程

利用上述终极秒杀技巧，这三道题根本不需要去列哪怕一步的差商表。

**第 (1) 题**
已知 $f(x) = 4x^2 + x - 3$。
这是一个 **2 次**多项式，最高次项系数 $a_2 = 4$。
- $f[x_n, x_{n+1}, x_{n+2}]$ 含有 3 个节点，是**二阶差商**。根据规律，2次多项式的二阶差商等于最高项系数，所以值为 **4**。
- $f[x_n, x_{n+1}, x_{n+2}, x_{n+3}]$ 含有 4 个节点，是**三阶差商**。因为 $3 > 2$，高于多项式的次数，所以值为 **0**。

**第 (2) 题**
已知 $f(x) = x^7 + x^4 + 3x + 1$。
这是一个 **7 次**多项式，最高次项系数 $a_7 = 1$。节点看起来很吓人（是 $2$ 的幂次），但根本不需要算。
- $f[2^0, 2^1, \dots, 2^7]$ 从 $2^0$ 到 $2^7$ 共有 8 个点，它是**七阶差商**。7 次多项式的七阶差商等于最高次项系数，值为 **1**。
- $f[2^0, 2^1, \dots, 2^8]$ 从 $2^0$ 到 $2^8$ 共有 9 个点，是**八阶差商**。因为 $8 > 7$，阶数一旦越界，全部是 **0**。

**第 (3) 题**
已知 $f(x) = 3x^2 + 1$。
这是一个 **2 次**多项式，最高次项系数 $a_2 = 3$。
- $f[1, 2, 3]$ 含有 3 个节点，是**二阶差商**。等于最高项系数，所以值为 **3**。
- $f[1, 2, 3, 4]$ 含有 4 个节点，是**三阶差商**。由于 $3 > 2$，所以值为 **0**。

---

## 最终答案
(1) `4` ; `0`
(2) `1` ; `0`
(3) `3` ; `0`
