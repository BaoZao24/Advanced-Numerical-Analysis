# 2.2 插值基函数答案

设 `x_i (i=0,1,2,3)` 为互异节点，`l_i(x)` 为对应的三次插值基函数。

利用 Lagrange 插值性质：若 `f(x)` 是次数不超过 `3` 的多项式，则

```math
\sum_{i=0}^{3} f(x_i)l_i(x)=f(x)
```

---

## （1）

```math
\sum_{i=0}^{3} x_i^3l_i(0)
```

取 `f(x)=x^3`，则

```math
\sum_{i=0}^{3} x_i^3l_i(x)=x^3
```

令 `x=0`，得

```math
\sum_{i=0}^{3} x_i^3l_i(0)=0
```

答案：

```math
\boxed{0}
```

---

## （2）

```math
\sum_{i=0}^{3}(x_i^3+2x_i-5)l_i(x)
```

取 `f(x)=x^3+2x-5`，这是三次多项式，因此插值精确：

```math
\sum_{i=0}^{3}(x_i^3+2x_i-5)l_i(x)=x^3+2x-5
```

答案：

```math
\boxed{x^3+2x-5}
```

---

## （3）

当 `k=0,1,2,3` 时，求

```math
\sum_{i=0}^{3}(x_i-x)^k l_i(x)
```

取

```math
f(t)=(t-x)^k
```

当 `k=0,1,2,3` 时，`f(t)` 关于 `t` 的次数不超过 `3`，所以

```math
\sum_{i=0}^{3}(x_i-x)^k l_i(x)=f(x)=(x-x)^k
```

因此

```math
\sum_{i=0}^{3}(x_i-x)^k l_i(x)=
\begin{cases}
1, & k=0,\\
0, & k=1,2,3.
\end{cases}
```

答案：

```math
\boxed{
\begin{cases}
1, & k=0,\\
0, & k=1,2,3.
\end{cases}}
```

---

## （4）

推广到一般式：

```math
\sum_{i=0}^{n}(x_i-x)^k l_i(x),\quad k=0,1,\cdots,n
```

同理，取

```math
f(t)=(t-x)^k
```

因为 `k\le n`，所以插值精确：

```math
\sum_{i=0}^{n}(x_i-x)^k l_i(x)=f(x)=(x-x)^k
```

因此

```math
\sum_{i=0}^{n}(x_i-x)^k l_i(x)=
\begin{cases}
1, & k=0,\\
0, & k=1,2,\cdots,n.
\end{cases}
```

答案：

```math
\boxed{
\begin{cases}
1, & k=0,\\
0, & k=1,2,\cdots,n.
\end{cases}}
```

---

## （5）

```math
\sum_{i=0}^{n}
\left[
\prod_{\substack{j=0\\j\ne i}}^{n}
\frac{x-j}{i-j}
\right] i
```

节点为

```math
0,1,2,\cdots,n
```

括号中的部分正是节点 `i` 对应的 Lagrange 插值基函数：

```math
l_i(x)=
\prod_{\substack{j=0\\j\ne i}}^{n}
\frac{x-j}{i-j}
```

所以原式为

```math
\sum_{i=0}^{n} i l_i(x)
```

这是函数 `f(t)=t` 在节点 `0,1,2,\cdots,n` 上的插值多项式。由于 `f(t)=t` 是一次多项式，插值精确，因此

```math
\sum_{i=0}^{n} i l_i(x)=x
```

答案：

```math
\boxed{x}
```
