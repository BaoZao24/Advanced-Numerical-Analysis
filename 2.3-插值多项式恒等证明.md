# 2.3 插值多项式恒等证明

## 题目

设 `f(x)` 是 `n` 次多项式，试证明 `f(x)` 在 `n+1` 个互异节点

```math
x_0,x_1,\cdots,x_n
```

上的次数不超过 `n` 的插值多项式 `p(x)` 恒等于 `f(x)`。

即证明：

```math
p(x) \equiv f(x)
```

---

## 用到的知识点

### 1. 插值多项式的定义

若多项式 `p(x)` 满足

```math
p(x_i)=f(x_i),\quad i=0,1,\cdots,n
```

则称 `p(x)` 是函数 `f(x)` 在节点 `x_0,x_1,\cdots,x_n` 上的插值多项式。

---

### 2. 插值多项式的唯一性

在 `n+1` 个互异节点上，次数不超过 `n` 的插值多项式是唯一的。

也就是说，如果两个次数不超过 `n` 的多项式在 `n+1` 个互异节点上函数值都相同，那么这两个多项式必定恒等相等。

---

### 3. 多项式零点定理

一个次数不超过 `n` 的非零多项式，最多只有 `n` 个不同零点。

因此，如果一个次数不超过 `n` 的多项式有 `n+1` 个互异零点，那么它只能是零多项式。

---

## 解题思路

要证明

```math
p(x) \equiv f(x)
```

可以构造差多项式：

```math
q(x)=f(x)-p(x)
```

因为 `f(x)` 是 `n` 次多项式，`p(x)` 是次数不超过 `n` 的插值多项式，所以 `q(x)` 的次数也不超过 `n`。

又因为 `p(x)` 是 `f(x)` 在节点 `x_0,x_1,\cdots,x_n` 上的插值多项式，所以有

```math
p(x_i)=f(x_i),\quad i=0,1,\cdots,n
```

于是

```math
q(x_i)=f(x_i)-p(x_i)=0,\quad i=0,1,\cdots,n
```

这说明 `q(x)` 在 `n+1` 个互异节点上都有零点。

但是 `q(x)` 的次数不超过 `n`，如果它不是零多项式，最多只能有 `n` 个互异零点。

因此只能有

```math
q(x)\equiv 0
```

即

```math
f(x)-p(x)\equiv 0
```

所以

```math
p(x)\equiv f(x)
```

---

## 证明过程

因为 `p(x)` 是 `f(x)` 在 `n+1` 个互异节点 `x_0,x_1,\cdots,x_n` 上的插值多项式，所以

```math
p(x_i)=f(x_i),\quad i=0,1,\cdots,n
```

令

```math
q(x)=f(x)-p(x)
```

由于 `f(x)` 是 `n` 次多项式，`p(x)` 的次数不超过 `n`，所以

```math
\deg q(x)\le n
```

并且对每一个节点 `x_i`，都有

```math
q(x_i)=f(x_i)-p(x_i)=0,\quad i=0,1,\cdots,n
```

因此 `q(x)` 有 `n+1` 个互异零点。

根据多项式零点定理，次数不超过 `n` 的非零多项式最多只有 `n` 个互异零点。

所以 `q(x)` 只能是零多项式，即

```math
q(x)\equiv 0
```

于是

```math
f(x)-p(x)\equiv 0
```

即

```math
p(x)\equiv f(x)
```

故原命题成立。

---

## 结论

若 `f(x)` 本身就是次数不超过 `n` 的多项式，则它在任意 `n+1` 个互异节点上的次数不超过 `n` 的插值多项式就是它本身。

最终结论：

```math
\boxed{p(x)\equiv f(x)}
```
