# 2.4 函数插值求一次插值多项式

## 题目

利用函数插值，求函数

```math
f(x)=e^{-x}
```

在区间 `[0,1]` 上的近似一次多项式 `p_1(x)`，要求：

1. 用待定系数法；
2. 用 Lagrange 插值公式；
3. 根据多项式插值问题的定义和定理直接“拼写”出来。

---

## 用到的知识点

### 1. 一次插值多项式

在两个互异节点 `x_0,x_1` 上，函数 `f(x)` 的一次插值多项式 `p_1(x)` 满足

```math
p_1(x_0)=f(x_0),\quad p_1(x_1)=f(x_1)
```

并且 `p_1(x)` 的次数不超过 `1`。

---

### 2. 待定系数法

设一次多项式为

```math
p_1(x)=a_0+a_1x
```

再利用插值条件列方程求出 `a_0,a_1`。

---

### 3. Lagrange 插值公式

两个节点 `x_0,x_1` 上的一次 Lagrange 插值公式为

```math
p_1(x)=f(x_0)l_0(x)+f(x_1)l_1(x)
```

其中

```math
l_0(x)=\frac{x-x_1}{x_0-x_1},\quad
l_1(x)=\frac{x-x_0}{x_1-x_0}
```

---

### 4. 插值多项式的存在唯一性

在 `n+1` 个互异节点上，次数不超过 `n` 的插值多项式存在且唯一。

本题中有两个节点 `0` 和 `1`，所以存在唯一的一次插值多项式。

---

## 解题准备

因为要求 `f(x)=e^{-x}` 在 `[0,1]` 上的一次插值多项式，所以取区间端点作为插值节点：

```math
x_0=0,\quad x_1=1
```

函数值为

```math
f(0)=e^0=1
```

```math
f(1)=e^{-1}=\frac{1}{e}
```

所以插值条件是

```math
p_1(0)=1,\quad p_1(1)=e^{-1}
```

---

# 方法一：待定系数法

设一次插值多项式为

```math
p_1(x)=a_0+a_1x
```

由插值条件 `p_1(0)=1`，得

```math
p_1(0)=a_0=1
```

所以

```math
a_0=1
```

由插值条件 `p_1(1)=e^{-1}`，得

```math
p_1(1)=a_0+a_1=e^{-1}
```

代入 `a_0=1`，得

```math
1+a_1=e^{-1}
```

所以

```math
a_1=e^{-1}-1
```

因此

```math
p_1(x)=1+(e^{-1}-1)x
```

答案为

```math
\boxed{p_1(x)=1+(e^{-1}-1)x}
```

也可以写成

```math
\boxed{p_1(x)=1-\left(1-e^{-1}\right)x}
```

---

# 方法二：Lagrange 插值公式

取节点

```math
x_0=0,\quad x_1=1
```

对应函数值为

```math
f(x_0)=f(0)=1
```

```math
f(x_1)=f(1)=e^{-1}
```

一次 Lagrange 插值公式为

```math
p_1(x)=f(x_0)l_0(x)+f(x_1)l_1(x)
```

其中

```math
l_0(x)=\frac{x-x_1}{x_0-x_1}
=\frac{x-1}{0-1}
=1-x
```

```math
l_1(x)=\frac{x-x_0}{x_1-x_0}
=\frac{x-0}{1-0}
=x
```

因此

```math
p_1(x)=1\cdot(1-x)+e^{-1}\cdot x
```

整理得

```math
p_1(x)=1-x+e^{-1}x
```

即

```math
p_1(x)=1+(e^{-1}-1)x
```

答案为

```math
\boxed{p_1(x)=1+(e^{-1}-1)x}
```

---

# 方法三：根据插值问题定义和定理直接写出

一次插值多项式就是通过两点

```math
(0,f(0)),\quad (1,f(1))
```

的直线。

因为

```math
f(0)=1,\quad f(1)=e^{-1}
```

所以这条直线通过两点

```math
(0,1),\quad (1,e^{-1})
```

直线的斜率为

```math
k=\frac{e^{-1}-1}{1-0}=e^{-1}-1
```

又因为直线过点 `(0,1)`，所以直线方程为

```math
p_1(x)=1+(e^{-1}-1)x
```

因此直接得到

```math
\boxed{p_1(x)=1+(e^{-1}-1)x}
```

---

# 最终答案

函数

```math
f(x)=e^{-x}
```

在 `[0,1]` 上，以端点 `0,1` 为插值节点的一次插值多项式为

```math
\boxed{p_1(x)=1+(e^{-1}-1)x}
```

或写成

```math
\boxed{p_1(x)=1-\left(1-e^{-1}\right)x}
```

---

# 小结

本题的核心是：

```math
p_1(0)=f(0),\quad p_1(1)=f(1)
```

即

```math
p_1(0)=1,\quad p_1(1)=e^{-1}
```

所以 `p_1(x)` 就是连接 `(0,1)` 和 `(1,e^{-1})` 的直线。
