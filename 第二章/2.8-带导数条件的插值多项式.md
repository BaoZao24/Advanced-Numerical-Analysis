# 2.8 带导数条件的插值多项式

## 题目

试用较简捷的方法求解：

### （1）

求满足

```math
p(0)=1,\quad p(1)=p'(1)=0,\quad p(2)=2
```

的插值多项式 `p(x)`。

### （2）

求满足

```math
p(0)=p'(0)=0,\quad p(1)=p'(1)=1,\quad p(2)=1
```

的插值多项式 `p(x)`。

---

## 这题要做什么

这道题和普通 Lagrange 插值不完全一样。

普通插值通常只给函数值，比如：

```math
p(0),\quad p(1),\quad p(2)
```

但本题还给了导数条件，比如：

```math
p'(1)=0
```

或者

```math
p'(0)=0,\quad p'(1)=1
```

这种既给函数值，又给导数值的插值问题，属于 **Hermite 插值问题**。

不过这道题不需要直接套复杂的 Hermite 插值公式，可以用更简捷的方法：

利用 **零点重数** 来构造多项式。

---

## 用到的知识点

### 1. 插值多项式的次数判断

如果一共有 `m` 个插值条件，一般可以构造次数不超过 `m-1` 的插值多项式。

例如第（1）问有 4 个条件：

```math
p(0)=1,\quad p(1)=0,\quad p'(1)=0,\quad p(2)=2
```

所以一般考虑次数不超过 `3` 的多项式。

第（2）问有 5 个条件：

```math
p(0)=0,\quad p'(0)=0,\quad p(1)=1,\quad p'(1)=1,\quad p(2)=1
```

所以一般考虑次数不超过 `4` 的多项式。

---

### 2. 零点和重零点

如果

```math
p(a)=0
```

说明 `x=a` 是 `p(x)` 的一个零点，因此 `p(x)` 中含有因子

```math
x-a
```

如果同时满足

```math
p(a)=0,\quad p'(a)=0
```

说明 `x=a` 是 `p(x)` 的二重零点，因此 `p(x)` 中含有因子

```math
(x-a)^2
```

这是本题最关键的知识点。

---

### 3. 用因式分解快速构造多项式

如果题目告诉我们某一点既满足函数值为 `0`，又满足导数值为 `0`，就可以直接写出平方因子。

例如：

```math
p(1)=0,\quad p'(1)=0
```

说明 `x=1` 是二重零点，所以

```math
p(x)=(x-1)^2q(x)
```

这样可以大大减少计算量。

---

# 第（1）问

## 条件

要求

```math
p(0)=1,\quad p(1)=p'(1)=0,\quad p(2)=2
```

也就是：

```math
p(0)=1
```

```math
p(1)=0
```

```math
p'(1)=0
```

```math
p(2)=2
```

---

## 解题思路

因为

```math
p(1)=0,\quad p'(1)=0
```

所以 `x=1` 是 `p(x)` 的二重零点。

因此 `p(x)` 中一定含有因子

```math
(x-1)^2
```

第（1）问共有 4 个条件，所以设 `p(x)` 是次数不超过 `3` 的多项式。

既然已经有二次因子 `(x-1)^2`，那么还需要乘一个一次多项式。

设

```math
p(x)=(x-1)^2(ax+b)
```

---

## 代入条件求系数

由条件 `p(0)=1`，得

```math
p(0)=(0-1)^2(a\cdot 0+b)=b
```

所以

```math
b=1
```

由条件 `p(2)=2`，得

```math
p(2)=(2-1)^2(2a+b)=2a+b
```

所以

```math
2a+b=2
```

代入 `b=1`，得

```math
2a+1=2
```

所以

```math
a=\frac{1}{2}
```

因此

```math
p(x)=(x-1)^2\left(\frac{1}{2}x+1\right)
```

也可以写成

```math
p(x)=\frac{1}{2}x^3-\frac{3}{2}x+1
```

---

## 第（1）问答案

```math
\boxed{p(x)=(x-1)^2\left(\frac{1}{2}x+1\right)}
```

或

```math
\boxed{p(x)=\frac{1}{2}x^3-\frac{3}{2}x+1}
```

---

# 第（2）问

## 条件

要求

```math
p(0)=p'(0)=0,\quad p(1)=p'(1)=1,\quad p(2)=1
```

也就是：

```math
p(0)=0
```

```math
p'(0)=0
```

```math
p(1)=1
```

```math
p'(1)=1
```

```math
p(2)=1
```

---

## 解题思路

因为

```math
p(0)=0,\quad p'(0)=0
```

所以 `x=0` 是 `p(x)` 的二重零点。

因此 `p(x)` 中一定含有因子

```math
x^2
```

第（2）问共有 5 个条件，所以设 `p(x)` 是次数不超过 `4` 的多项式。

既然已经有二次因子 `x^2`，剩下部分可以设成二次多项式：

```math
p(x)=x^2(ax^2+bx+c)
```

即

```math
p(x)=ax^4+bx^3+cx^2
```

---

## 代入条件求系数

先求导：

```math
p'(x)=4ax^3+3bx^2+2cx
```

由条件 `p(1)=1`，得

```math
a+b+c=1
```

由条件 `p'(1)=1`，得

```math
4a+3b+2c=1
```

由条件 `p(2)=1`，得

```math
16a+8b+4c=1
```

于是得到方程组：

```math
\begin{cases}
a+b+c=1,\\
4a+3b+2c=1,\\
16a+8b+4c=1.
\end{cases}
```

---

## 解方程组

由第一式

```math
a+b+c=1
```

两边乘以 `2`，得

```math
2a+2b+2c=2
```

用第二式减去它：

```math
(4a+3b+2c)-(2a+2b+2c)=1-2
```

得到

```math
2a+b=-1
```

再由第一式两边乘以 `4`，得

```math
4a+4b+4c=4
```

用第三式减去它：

```math
(16a+8b+4c)-(4a+4b+4c)=1-4
```

得到

```math
12a+4b=-3
```

即

```math
3a+b=-\frac{3}{4}
```

于是有

```math
\begin{cases}
2a+b=-1,\\
3a+b=-\frac{3}{4}.
\end{cases}
```

两式相减，得

```math
a=\frac{1}{4}
```

代入

```math
2a+b=-1
```

得

```math
2\cdot\frac{1}{4}+b=-1
```

所以

```math
b=-\frac{3}{2}
```

再代入

```math
a+b+c=1
```

得

```math
\frac{1}{4}-\frac{3}{2}+c=1
```

所以

```math
c=\frac{9}{4}
```

因此

```math
p(x)=x^2\left(\frac{1}{4}x^2-\frac{3}{2}x+\frac{9}{4}\right)
```

也就是

```math
p(x)=\frac{1}{4}x^4-\frac{3}{2}x^3+\frac{9}{4}x^2
```

---

## 第（2）问答案

```math
\boxed{p(x)=x^2\left(\frac{1}{4}x^2-\frac{3}{2}x+\frac{9}{4}\right)}
```

或

```math
\boxed{p(x)=\frac{1}{4}x^4-\frac{3}{2}x^3+\frac{9}{4}x^2}
```

---

# 最终答案汇总

## （1）

```math
\boxed{p(x)=(x-1)^2\left(\frac{1}{2}x+1\right)}
```

即

```math
\boxed{p(x)=\frac{1}{2}x^3-\frac{3}{2}x+1}
```

## （2）

```math
\boxed{p(x)=x^2\left(\frac{1}{4}x^2-\frac{3}{2}x+\frac{9}{4}\right)}
```

即

```math
\boxed{p(x)=\frac{1}{4}x^4-\frac{3}{2}x^3+\frac{9}{4}x^2}
```

---

# 这类题怎么做

遇到这种带导数条件的插值题，可以按下面步骤做。

## 第一步：看有没有函数值和导数值同时为 0

如果有：

```math
p(a)=0,\quad p'(a)=0
```

就立刻写出：

```math
p(x)=(x-a)^2q(x)
```

这是最快的突破口。

## 第二步：判断多项式最高次数

数一共有几个条件。

如果有 `m` 个条件，通常设次数不超过 `m-1`。

## 第三步：根据已知零点因子设多项式

比如：

```math
p(1)=p'(1)=0
```

就设：

```math
p(x)=(x-1)^2(ax+b)
```

如果：

```math
p(0)=p'(0)=0
```

就设：

```math
p(x)=x^2(ax^2+bx+c)
```

## 第四步：代入剩下条件求系数

把还没用的条件代入，列方程求出未知系数。

## 第五步：检查条件

最后可以快速检查：

- 函数值条件是否满足；
- 导数条件是否满足；
- 次数是否符合要求。

---

# 本题最关键的一句话

如果

```math
p(a)=0,\quad p'(a)=0
```

那么 `x=a` 是二重零点，所以 `p(x)` 含有因子

```math
(x-a)^2
```

这就是这道题“较简捷方法”的核心。
