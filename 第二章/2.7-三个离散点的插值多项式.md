# 2.7 三个离散点的插值多项式

## 题目

用多种解法，至少三种或以上，求过三个离散点

```math
A(0,1),\quad B(1,2),\quad C(2,3)
```

的插值多项式。

---

## 这题要做什么

题目给了三个点：

```math
(0,1),\quad (1,2),\quad (2,3)
```

要求一个多项式 `p(x)`，使它满足：

```math
p(0)=1
```

```math
p(1)=2
```

```math
p(2)=3
```

因为有 `3` 个互异节点，所以一般可以构造一个次数不超过 `2` 的插值多项式。

也就是说，我们要求的是一个二次或二次以下的多项式：

```math
p_2(x)
```

使它通过这三个点。

---

## 用到的知识点

### 1. 插值多项式的定义

给定 `n+1` 个互异节点

```math
x_0,x_1,\cdots,x_n
```

以及对应函数值

```math
y_0,y_1,\cdots,y_n
```

如果多项式 `p_n(x)` 满足

```math
p_n(x_i)=y_i,\quad i=0,1,\cdots,n
```

则称 `p_n(x)` 为这些节点上的插值多项式。

本题中有三个点，所以 `n=2`，插值多项式次数不超过 `2`。

---

### 2. 插值多项式的唯一性

通过 `n+1` 个互异节点的次数不超过 `n` 的插值多项式是唯一的。

本题中，通过三个互异节点的次数不超过 `2` 的插值多项式唯一。

所以不管用哪种方法，最后结果都应该相同。

---

### 3. 待定系数法

设

```math
p_2(x)=a_0+a_1x+a_2x^2
```

再代入三个点，列方程求 `a_0,a_1,a_2`。

---

### 4. Lagrange 插值公式

三个节点的一般 Lagrange 插值公式为

```math
p_2(x)=y_0l_0(x)+y_1l_1(x)+y_2l_2(x)
```

其中

```math
l_i(x)=\prod_{j=0,j\ne i}^{2}\frac{x-x_j}{x_i-x_j}
```

---

### 5. Newton 插值公式

Newton 插值法也是一种构造插值多项式的方法。

它和 Lagrange 插值法的结果完全一样，只是写法不同。

Lagrange 插值法是把多项式写成几个基函数的加权和；Newton 插值法是把多项式写成下面这种逐层增加的形式：

```math
p_2(x)=a_0+a_1(x-x_0)+a_2(x-x_0)(x-x_1)
```

这里：

- `a_0` 由第一个点决定；
- `a_1` 由前两个点共同决定；
- `a_2` 由前三个点共同决定。

这些系数不是随便取的，它们叫做差商。

对于三个节点，Newton 插值公式为

```math
p_2(x)=f[x_0]+f[x_0,x_1](x-x_0)+f[x_0,x_1,x_2](x-x_0)(x-x_1)
```

其中 `f[x_i]` 是函数值，`f[x_i,x_j]` 是一阶差商，`f[x_i,x_j,x_k]` 是二阶差商。

一阶差商的公式是：

```math
f[x_i,x_j]=\frac{f(x_j)-f(x_i)}{x_j-x_i}
```

它其实就是两点之间的斜率。

二阶差商的公式是：

```math
f[x_0,x_1,x_2]=\frac{f[x_1,x_2]-f[x_0,x_1]}{x_2-x_0}
```

它反映的是斜率变化的情况。

如果二阶差商为 `0`，说明斜率没有变化，这几个点在同一条直线上，最后得到的插值多项式就是一次多项式。

---

# 方法一：待定系数法

设二次插值多项式为

```math
p_2(x)=a_0+a_1x+a_2x^2
```

由点 `A(0,1)`，得

```math
p_2(0)=a_0=1
```

所以

```math
a_0=1
```

由点 `B(1,2)`，得

```math
p_2(1)=a_0+a_1+a_2=2
```

代入 `a_0=1`，得

```math
a_1+a_2=1
```

由点 `C(2,3)`，得

```math
p_2(2)=a_0+2a_1+4a_2=3
```

代入 `a_0=1`，得

```math
2a_1+4a_2=2
```

即

```math
a_1+2a_2=1
```

于是得到方程组：

```math
\begin{cases}
a_1+a_2=1,\\
a_1+2a_2=1.
\end{cases}
```

两式相减，得

```math
a_2=0
```

所以

```math
a_1=1
```

因此

```math
p_2(x)=1+x
```

即

```math
\boxed{p_2(x)=x+1}
```

---

# 方法二：Lagrange 插值法

三个节点为

```math
x_0=0,\quad x_1=1,\quad x_2=2
```

对应函数值为

```math
y_0=1,\quad y_1=2,\quad y_2=3
```

Lagrange 插值多项式为

```math
p_2(x)=y_0l_0(x)+y_1l_1(x)+y_2l_2(x)
```

其中

```math
l_0(x)=\frac{x-1}{0-1}\cdot\frac{x-2}{0-2}
```

```math
l_0(x)=\frac{(x-1)(x-2)}{2}
```

```math
l_1(x)=\frac{x-0}{1-0}\cdot\frac{x-2}{1-2}
```

```math
l_1(x)=-x(x-2)
```

```math
l_2(x)=\frac{x-0}{2-0}\cdot\frac{x-1}{2-1}
```

```math
l_2(x)=\frac{x(x-1)}{2}
```

所以

```math
p_2(x)=1\cdot\frac{(x-1)(x-2)}{2}+2\cdot[-x(x-2)]+3\cdot\frac{x(x-1)}{2}
```

展开计算：

```math
p_2(x)=\frac{x^2-3x+2}{2}-2x^2+4x+\frac{3x^2-3x}{2}
```

合并同类项：

```math
p_2(x)=\frac{x^2+3x^2}{2}-2x^2+\frac{-3x-3x}{2}+4x+1
```

```math
p_2(x)=2x^2-2x^2-3x+4x+1
```

```math
p_2(x)=x+1
```

因此

```math
\boxed{p_2(x)=x+1}
```

---

# 方法三：Newton 插值法

## 1. 先理解 Newton 插值法在做什么

本题有三个点，所以按道理可以构造一个次数不超过 `2` 的插值多项式。

Newton 插值法不直接写成

```math
p_2(x)=a_0+a_1x+a_2x^2
```

而是写成：

```math
p_2(x)=a_0+a_1(x-x_0)+a_2(x-x_0)(x-x_1)
```

为什么这样写？

因为这种形式有一个好处：每增加一个新点，只需要在后面加一项，不用推翻前面的结果。

比如：

只用第一个点时：

```math
p_0(x)=a_0
```

再加入第二个点时：

```math
p_1(x)=a_0+a_1(x-x_0)
```

再加入第三个点时：

```math
p_2(x)=a_0+a_1(x-x_0)+a_2(x-x_0)(x-x_1)
```

这就是 Newton 插值法的基本思想：从低次到高次，一层一层加项。

---

## 2. 本题的节点和函数值

取节点：

```math
x_0=0,\quad x_1=1,\quad x_2=2
```

函数值：

```math
f(x_0)=1,\quad f(x_1)=2,\quad f(x_2)=3
```

这里的 `f(x_0),f(x_1),f(x_2)` 其实就是三个点的纵坐标。

也就是：

```math
f(0)=1,\quad f(1)=2,\quad f(2)=3
```

---

## 3. 计算一阶差商

一阶差商可以理解成“两点之间的斜率”。

先计算一阶差商：

```math
f[x_0,x_1]=\frac{f(x_1)-f(x_0)}{x_1-x_0}
```

```math
f[x_0,x_1]=\frac{2-1}{1-0}=1
```

```math
f[x_1,x_2]=\frac{f(x_2)-f(x_1)}{x_2-x_1}
```

```math
f[x_1,x_2]=\frac{3-2}{2-1}=1
```

这说明：

从点 `A(0,1)` 到点 `B(1,2)` 的斜率是 `1`；

从点 `B(1,2)` 到点 `C(2,3)` 的斜率也是 `1`。

也就是说，这三点的斜率没有变化，很可能在同一条直线上。

---

## 4. 计算二阶差商

二阶差商可以理解成“斜率的变化率”。

再计算二阶差商：

```math
f[x_0,x_1,x_2]=\frac{f[x_1,x_2]-f[x_0,x_1]}{x_2-x_0}
```

```math
f[x_0,x_1,x_2]=\frac{1-1}{2-0}=0
```

因为两个一阶差商都等于 `1`，所以它们的差为 `0`。

因此二阶差商为 `0`。

这说明二次项的系数是 `0`，最终多项式其实不是二次的，而是一次的。

---

## 5. 写出 Newton 插值公式

Newton 插值公式为

```math
p_2(x)=f[x_0]+f[x_0,x_1](x-x_0)+f[x_0,x_1,x_2](x-x_0)(x-x_1)
```

其中：

```math
f[x_0]=f(x_0)=1
```

```math
f[x_0,x_1]=1
```

```math
f[x_0,x_1,x_2]=0
```

代入数据：

```math
p_2(x)=1+1\cdot(x-0)+0\cdot(x-0)(x-1)
```

所以

```math
p_2(x)=1+x
```

即

```math
\boxed{p_2(x)=x+1}
```

---

## 6. Newton 插值法的计算表

为了更直观，可以把差商写成表格。

| 节点 | 函数值 | 一阶差商 | 二阶差商 |
|---|---:|---:|---:|
| `x_0=0` | `f(x_0)=1` |  |  |
| `x_1=1` | `f(x_1)=2` | `f[x_0,x_1]=1` |  |
| `x_2=2` | `f(x_2)=3` | `f[x_1,x_2]=1` | `f[x_0,x_1,x_2]=0` |

Newton 插值多项式取的是表格中从左上角开始的一条“斜线”上的系数：

```math
f[x_0],\quad f[x_0,x_1],\quad f[x_0,x_1,x_2]
```

也就是：

```math
1,\quad 1,\quad 0
```

所以：

```math
p_2(x)=1+1(x-0)+0(x-0)(x-1)
```

即：

```math
p_2(x)=x+1
```

---

## 7. 初学者记法

如果你刚开始学 Newton 插值法，可以先记住三个步骤。

### 第一步：写 Newton 形式

三个点时写：

```math
p_2(x)=f[x_0]+f[x_0,x_1](x-x_0)+f[x_0,x_1,x_2](x-x_0)(x-x_1)
```

### 第二步：算差商

先算一阶差商：

```math
f[x_0,x_1],\quad f[x_1,x_2]
```

再算二阶差商：

```math
f[x_0,x_1,x_2]
```

### 第三步：代入公式

把算出来的差商放进 Newton 公式里，然后化简。

本题中：

```math
f[x_0]=1,\quad f[x_0,x_1]=1,\quad f[x_0,x_1,x_2]=0
```

所以：

```math
p_2(x)=1+1(x-0)+0(x-0)(x-1)=x+1
```

# 方法四：直接观察法

观察三个点：

```math
(0,1),\quad (1,2),\quad (2,3)
```

可以发现它们都满足关系：

```math
y=x+1
```

验证：

当 `x=0` 时，

```math
y=0+1=1
```

当 `x=1` 时，

```math
y=1+1=2
```

当 `x=2` 时，

```math
y=2+1=3
```

所以这三个点都在直线

```math
y=x+1
```

上。

因此插值多项式可以直接写为

```math
p_2(x)=x+1
```

由于插值多项式唯一，所以这个结果就是所求插值多项式。

---

# 最终答案

过三个离散点

```math
A(0,1),\quad B(1,2),\quad C(2,3)
```

的插值多项式为

```math
\boxed{p_2(x)=x+1}
```

虽然题目中有三个点，一般求的是次数不超过 `2` 的插值多项式，但由于这三个点恰好共线，所以二次项系数为 `0`，最终得到的是一次多项式。

---

## 这类题怎么做

遇到“给几个离散点，求插值多项式”的题，可以按下面步骤做：

### 第一步：确定节点和函数值

把点写成

```math
(x_0,y_0),(x_1,y_1),\cdots,(x_n,y_n)
```

本题中：

```math
x_0=0,\quad y_0=1
```

```math
x_1=1,\quad y_1=2
```

```math
x_2=2,\quad y_2=3
```

### 第二步：判断需要几次多项式

有 `n+1` 个点，就构造次数不超过 `n` 的插值多项式。

本题有 `3` 个点，所以构造次数不超过 `2` 的多项式。

### 第三步：选择方法

常用方法有：

1. 待定系数法；
2. Lagrange 插值法；
3. Newton 插值法；
4. 如果点有明显规律，也可以直接观察。

### 第四步：代入并化简

不管用哪种方法，最后都要化简多项式。

本题最后化简得到：

```math
p_2(x)=x+1
```
