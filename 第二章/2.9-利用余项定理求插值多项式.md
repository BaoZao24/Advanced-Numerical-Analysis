# 2.9 利用 Lagrange 插值余项定理求插值多项式

## 题目要求
设 $f(x)=x^4$，不直接用 Lagrange 插值公式，而用 Lagrange 插值余项定理求出以 $x=-1, 0, 1, 2$ 为插值节点的三次插值多项式 $L_3(x)$。

---

## 知识点解析

### 1. 插值多项式与余项的关系
对于给定的函数 $f(x)$ 及其 $n+1$ 个互异的插值节点 $x_0, x_1, \dots, x_n$，其次数为 不超过 $n$ 的插值多项式 $L_n(x)$ 满足：
$$f(x) = L_n(x) + R_n(x)$$
其中 $R_n(x)$ 为插值余项（即插值误差）。这就是说，只要我们知道了原函数 $f(x)$ 和截断误差（余项 $R_n(x)$），我们就可以反求出插值多项式 $L_n(x)$。

### 2. Lagrange 插值余项定理
如果 $f(x)$ 在包含插值节点的区间内具有 $n+1$ 阶导数，则插值多项式 $L_n(x)$ 的截断误差（余项）可表示为：
$$R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!} \omega_{n+1}(x)$$
其中：
- $\xi$ 是依赖于 $x$ 且位于插值节点和 $x$ 所构成的区间内的一个未知值。
- $\omega_{n+1}(x) = (x - x_0)(x - x_1)\dots(x - x_n)$ 是节点多项式。

### 3. 解题技巧
当被插值函数 $f(x)$ 是一个 $n+1$ 次多项式，而要我们求 $n$ 次插值多项式时，它的 $n+1$ 阶导数 $f^{(n+1)}(x)$ 必然是一个**常数**。此时余项定理常常能极大化简运算，从而比直接套用拉格朗日基函数公式要快得多。

---

## 解题过程

**步骤 1：明确已知条件**
- 目标函数：$f(x) = x^4$
- 插值节点：$x_0 = -1, x_1 = 0, x_2 = 1, x_3 = 2$
- 节点个数为 4，因此插值多项式的次数 $n = 3$。

**步骤 2：写出余项表达式**
根据 Lagrange 插值余项定理：
$$R_3(x) = \frac{f^{(4)}(\xi)}{4!} (x - x_0)(x - x_1)(x - x_2)(x - x_3)$$
将节点代入得：
$$R_3(x) = \frac{f^{(4)}(\xi)}{4!} (x + 1)(x - 0)(x - 1)(x - 2)$$

**步骤 3：计算函数的高阶导数**
已知 $f(x) = x^4$，我们对其求导：
- $f'(x) = 4x^3$
- $f''(x) = 12x^2$
- $f'''(x) = 24x$
- $f^{(4)}(x) = 24$

可以发现，$f^{(4)}(x)$ 是一个常数 24。因此，对于任意的 $\xi$，$f^{(4)}(\xi) = 24$。

**步骤 4：计算精确的余项 $R_3(x)$**
将 $f^{(4)}(\xi) = 24$ 和 $4! = 24$ 代入余项公式：
$$R_3(x) = \frac{24}{24} x(x + 1)(x - 1)(x - 2)$$
$$R_3(x) = x(x^2 - 1)(x - 2)$$
展开得：
$$R_3(x) = x(x^3 - 2x^2 - x + 2) = x^4 - 2x^3 - x^2 + 2x$$

**步骤 5：求出插值多项式 $L_3(x)$**
由 $f(x) = L_3(x) + R_3(x)$ 可知：
$$L_3(x) = f(x) - R_3(x)$$
将 $f(x)$ 和 $R_3(x)$ 代入：
$$L_3(x) = x^4 - (x^4 - 2x^3 - x^2 + 2x)$$
$$L_3(x) = 2x^3 + x^2 - 2x$$

## 最终答案
以 $x=-1, 0, 1, 2$ 为插值节点的三次插值多项式为：
$$L_3(x) = 2x^3 + x^2 - 2x$$
